Js와 php가 비슷하다?
똑같은일을 웹브라우저에서 하는것과 서버에서 하는것의 차이정도

하나의 html로 여러정보를 표현하는것이 php로 할수있는일
php는 기본적으로 html과 똑같이 활동한다.
<?php   ?>로 php를 표시한다.
php에서 무언가를 출력할떄 echo를 사용한다.
	echo "Hello world";
	echo 10+10; 은 20이 찍힌다.? >> php가 인식해서 사용한 결과를 나타내는것이다.
php는 서버쪽에서 실행이되는 언어이므로 결과만이 넘어가서 웹브라우저에서 출력
	>> php 인터프리터라인 애가 처리(서버쪽)
Js는 웹브라우저에서 실행되는 언어이므로 코드 전문이 넘어가있음
	>> 웹브라우저에서 실행(클라이언트)
문자열 표시는 ""로 한다.
숫자는 ""을 쓰면 안됨
php에서 +는 숫자를 더할때 쓰는 연산자로 고정되어있기 때문에 숫자를 문자열로 표시해서 더하게되도 자동으로 숫자로 수정하여 더하게된다. 문자열 두개를 더하는 연산자는 .으로 "10"."10"은 1010이다.
php에서 에러가 나면 화면에는 아무것도 뜨지않는다.
php에러를 화면에 표시하는 방법
$name ="알케미"; < 이것이 php에서 변수를 만드는 방법임 변수명앞에 $표시가 필요
php에서 변수를 불러올때는 항상 $을 붙여서 사용한다.
